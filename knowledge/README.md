# Pheebs Core Knowledge Repository

This directory serves as the raw, unstructured knowledge engine of our sales team. Before writing code or prompting LLMs, we collect real-world conversations, sales insights, and playbooks here. 

Eventually, this raw data is compiled into **Sales DNA** rules.

## Directories
- `/Ashish/`: Discovery shortcuts and local spa onboarding tips.
- `/Robin/`: Onboarding templates, timeline objections, and enterprise spa tactics.
- `/Piyush/`: Aggressive discovery validation checks and monologue warnings.
- `/Kripali/`: Target dental/medical practice data and Ghost Call case studies.
- `/Customers/`: Transcripts from real business evaluations.
- `/Winning_Calls/`: Successful recordings and verbatim closing moments.
- `/Lost_Calls/`: Post-mortem analyses of lost opportunities.
