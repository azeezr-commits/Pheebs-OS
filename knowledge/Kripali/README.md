# Kripali's Sales Playbook Notes
- **Focus**: Dental practices, medical clinics, and call abandonment leaks.
- **Rule 1**: Teach them about "Ghost Calls". Dental receptionists are busy; if they don't answer, patients call the next clinic. Missed calls don't leave voicemails.
- **objections handled**: "We don't miss calls / Reception is always staffed".
